Data Query Language (DQL)

DQL is used to fetch the data from the database.

Commands

SELECT

Primarily used for querying data from one or more tables.

Commands
Explanation:

The SELECT command is the most fundamental and commonly used command in SQL. It is primarily used to retrieve data from one or more tables in a database. You specify which columns you want to retrieve from the table(s) and use the FROM clause to specify the table(s) from which to retrieve the data. Optionally, you can use the WHERE clause to specify conditions that filter the rows returned by the query based on certain criteria. The result of a SELECT query is a result set containing the selected columns' values that meet the specified conditions.

Example:
The result table is
SNAME SID SAGE SAREA SDEPT SSPOCKET
ashwin 101 19 annanagar aeronautical 750
